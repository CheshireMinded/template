# Service Security Questionnaire (Template)

Use this before adding a new third-party dependency.

---

## Basic Information

- Service name  
- Vendor  
- Purpose  

---

## Security Topics

### Data Handling

- What data is stored or processed?
- Is data encrypted at rest and in transit?

### Authentication

- OAuth? API keys? Certificates?
- MFA supported?

### Compliance

- SOC2? ISO27001? GDPR?

### Logging & Monitoring

- Does vendor support audit logs?
- How long are logs retained?

### Vulnerability Management

- Do they disclose CVEs?
- Do they have a bug bounty?

Assign:

- High
