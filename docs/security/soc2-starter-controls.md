# SOC2 / ISO27001 Starter Control Mapping (Template)

> This is only a learning reference.  
> Teams must define real controls for production environments.

---

## Access Control

- Role-based permissions documented  
- Least privilege enforced  
- Access reviewed quarterly  

---

## Change Management

- PR reviews required  
- CI checks required  
- Production deployments logged  

---

## Logical Security

- Secrets stored securely  
- No plaintext credentials  
- MFA required (where applicable)  

---

## System Monitoring

- Error logs captured  
- Audit trails preserved  
- Alerting thresholds defined  

---

## Data Handling

- HTTPS enforced  
- Data classified (public/internal/confidential)  
- Data retention policy defined
