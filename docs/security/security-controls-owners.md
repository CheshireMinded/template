# Security Controls Ownership Map (Template)

This document maps categories of security controls to responsible roles.

| Control Area | Primary Owner | Notes |
|--------------|---------------|-------|
| Application Security | Engineering | Input validation, authN/Z |
| Infrastructure Security | DevOps | K8s hardening, Terraform |
| CI/CD Security | DevOps | Secrets, SAST, supply chain |
| Identity & Access | Engineering + DevOps | Roles, permissions |
| Incident Response | Engineering | See IR template |
| Risk Management | Maintainers | Template only |

In real production systems, assign real individuals or teams.
