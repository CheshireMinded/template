# Security Review Cadence (Template)

| Review Type | Frequency | Purpose |
|-------------|-----------|---------|
| Dependency Review | Weekly | Catch vulnerable packages early |
| Threat Model Review | Quarterly | Validate architecture & risks |
| Security Docs Audit | Quarterly | Ensure docs remain up to date |
| Infra Config Review | Each Release | Check IAM, K8s, Terraform |
| Incident Simulation | Yearly | Validate IR readiness |

This cadence helps maintain long-term security hygiene.
